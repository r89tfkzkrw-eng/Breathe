# Breathe 🧘‍♀️

Простое веб-приложение для дыхательных практик

## Что внутри:
- **Знакомые техники:** Квадратное дыхание, 4-7-8, физиологический вздох.
- **Свой ритм:** Можно настроить под себя, тайминги сохранятся в браузере.
- **Дышащий звук:** Никаких зацикленных MP3 файлов. Алгоритмический эмбиент с реверберацией синтезируется прямо в браузере с помощью Web Audio API и отзывается на каждую фазу Вашего дыхания. 

## Как запустить:

```bash
npm install
npm run dev
```
